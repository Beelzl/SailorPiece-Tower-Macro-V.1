# SailorPiece-Tower-Macro-V.1

This macro is designed to work exclusively with the Sailor Piece Madara Update.
It requires both Anos and Strongest of Today V2 to be present in your inventory in order to function properly.

# Personal use
An image detection system has not been implemented, so please ensure that all required units are already in your inventory before using the macro.

Feel free to use it if you meet the requirements.
